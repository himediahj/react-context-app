# react-context-app
1. 바탕화면에 react-context-app 폴더 생성
2. Visual Studio Code 열고 생성 폴더 드래그
3. server / client 폴더 생성 후 터미널 열기 -> cd client -> npx create-react-app ./
4. 서버에서 터미널 -> npm install -> npm run start
5. client 터미널에서 npm install axios
